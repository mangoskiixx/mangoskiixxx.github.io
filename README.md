# Tyler The Creator Album Website

A responsive multi-page website showcasing Tyler The Creator's discography and providing contact information.

## Features

- **Home Page**: Hero section, featured album, and latest updates
- **Albums Page**: Complete discography with album details
- **Contact Page**: Contact form with validation
- **Responsive Design**: Mobile-friendly layout
- **JavaScript Features**: 
  - Mobile navigation toggle
  - Form validation
  - Interactive buttons
  - Dynamic date display

## Technologies Used

- HTML5 (semantic elements)
- CSS3 (Flexbox, Grid, responsive design)
- JavaScript (DOM manipulation, form validation)
- Git for version control
